# Wordified RegEx (Alpha/Beta State)
[Wordified RegEx](https://tblazewarriort.github.io/WordifiedRegEx/) is a WIP browser and standalone tool made in [Godot Engine](https://godotengine.org/) 3.6 for editing Regular Expressions online in a more human-friendly way.

Important Tips:
- All buttons have tooltips and editable lines have placeholders.
- RegEx descriptions will be simplified, for proper explanations please refer to your own relevant documentation.
- Please double-check any RegEx output from this for errors.
